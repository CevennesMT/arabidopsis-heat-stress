# Stats408-FinalProj-Yves-Vernooy

## Broad Description
Final Project for Stats408. Investigating heat trauma response across genetically modified and control plant varieties. Specifically investigate differential gene expression in Arabidopsis thaliana under heat stress conditions for wild-type and transgenic lines meant to improve heat truama "performance". This project will in part investigate the findings of a published paper, see project resources for more information.

## Webpage Link
https://cevennesmt.github.io/arabidopsis-heat-stress/

## Project Aims
- Broadly: Create a Quarto dashboard that explores and investigates a plant data set.
- Specifically: Explore the following questions.
  - Does the transgenic line out perform the wild-type variety under heat stress conditions
  - Identify other gene expressions in both the transgenic line and wild-type groups under heat shock
  - Build a pipeline for and visualize the data as we answer the other questions.
